## How to use this template

1. Copy this file to its target location and rename.
2. Fill in title, status, date, and the first change-log row.
3. Delete this "How to use this template" section.
4. Delete the TOC if the doc is short.
5. Specialized doc types (ADRs, runbooks) keep their own header
   fields (Date / Status / Deciders for ADRs; Owner / Severity / Last
   reviewed for runbooks) but still include an inline `## Change Log`
   section.
